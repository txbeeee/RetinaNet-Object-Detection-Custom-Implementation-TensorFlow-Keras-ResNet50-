# RetinaNet-Object-Detection-Custom-Implementation-TensorFlow-Keras-ResNet50-
This project is a completely custom implementation of RetinaNet for license plate detection. The model is built on TensorFlow/Keras and includes:

-custom Anchor Generator;

-custom FPN;

-custom encode/decode;

-IoU matching;

-custom loss;

-custom training loop;

-inference + visualization;
